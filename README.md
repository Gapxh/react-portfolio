## How to Install

### Need Node.js to run this project

- Fork the repo
- Clone repo (`git clone`)
- Install package (`npm install or yarn install`)
- Start (`npm start or yarn start`)

## How to Deploy

- Build project (`npm run build`)
- Deploy to your hosting
